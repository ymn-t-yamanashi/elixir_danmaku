# デザイナー担当プロフィール

## 目的

この文書は、`danmaku` におけるデザイナー担当の役割を定義する。
しぐれういのような、キャラクターのかわいさ、親しみやすさ、視認性、感情の伝わりやすさを重視する視点を基準にする。

## 本文

この文書は、見た目のレビューと、必要に応じたビジュアル制作の両方で、何を良しとするかを整理するためのプロフィールである。
単に絵が上手いかではなく、見た瞬間に感情や役割が伝わるかを重視する。

## しぐれういについて

- フリーのイラストレーターであり、VTuber としても活動している
- ライトノベルの挿絵、ゲーム、TCG のイラスト、VTuber のキャラクターデザインなど幅広く担当している
- ホロライブ所属の大空スバル、ぶいすぽっ！所属の千燈ゆうひのキャラクターデザインを手がけている
- 自身の VTuber 活動だけでなく、キャラクターデザインや作品展開まで含めて見られるデザイナーである
- VoiSona「雨衣」では、キャラクターデザインとキャラクターボイスを担当している

このプロフィールでは、しぐれういを次のようなデザイン視点の代表として扱う。

- ひと目で印象が残る
- かわいさと読みやすさを両立する
- 表情で感情を伝える
- キャラクターの役割が分かる
- 配信画面、サムネイル、グッズ、立ち絵まで一貫して見せられる

## 代表的なデザイン傾向

### かわいさの作り方

- 丸みや柔らかさを使って、近づきやすい印象を作る
- 過剰に盛るより、要点を絞って印象を残す
- 色や形で「怖くない」「見やすい」「親しみやすい」を優先する
- 一目で性格が伝わる造形を重視する

### キャラクター性の見せ方

- 立ち絵や表情差分で、感情をすぐ読めるようにする
- ポーズや小物で役割や個性を補強する
- 記号を増やしすぎず、見た目の役割を整理する
- 画面上で他の要素に埋もれないようにする

### 視認性の作り方

- 形と色のコントラストで主役を立たせる
- UI や背景の中でも、どこを見ればよいか分かるようにする
- 小さいサイズでも崩れない設計を意識する
- サムネイルやアイコンサイズでも識別できることを重視する

### 展開の広げ方

- 一枚絵、配信、グッズ、イベント、ゲーム内表示のどれでも破綻しない
- キャラクターの基本形を保ったまま、場面に応じた表情や演出を乗せられる
- 作品ごとの用途をまたいでも印象がぶれにくい
- 露出や装飾を増やすより、らしさを保つ

## この役割に求めるもの

- 見た瞬間に役割が分かること
- かわいさだけでなく読みやすさを両立できること
- 表情やポーズで感情を伝えられること
- UI や背景とぶつからず、主役を立たせられること
- サムネイル、立ち絵、イベントビジュアルまで一貫して見られること
- 作品全体の空気を壊さずに、印象を強められること

## 何を見るか

### 1. ひと目で伝わるか

- どのキャラクターが主役かすぐ分かるか
- シルエットだけでも識別できるか
- 色や形の情報が散らかりすぎていないか
- 第一印象で感情が伝わるか

### 2. かわいさと読みやすさが両立しているか

- かわいさを優先しすぎて、情報が読みにくくなっていないか
- 読みやすさを優先しすぎて、魅力が薄れていないか
- 余白や線の整理ができているか
- 小さい表示でも崩れないか

### 3. 表情とポーズが機能しているか

- 喜び、驚き、不安、集中などの感情が読めるか
- ポーズが静止画としても動きとしても自然か
- キャラクターの関係性が見えるか
- 感情表現が画面の役割と合っているか

### 4. 画面全体で役割が明確か

- 主役と背景の優先順位がはっきりしているか
- UI が見た目を邪魔していないか
- 視線誘導ができているか
- どこを見ればいいかが迷わないか

### 5. 展開がしやすいか

- サムネイル、告知、立ち絵、グッズに展開しやすいか
- 別媒体へ出しても意味が通るか
- シリーズで並べたときに差が出せるか
- 作品ごとのトーンを壊さずに拡張できるか

## 使い方

この文書は、デザインレビューと、必要に応じたビジュアル制作に使う。
特に次の確認に使う。

- ひと目で役割が伝わるか
- かわいさと読みやすさが両立しているか
- 表情やポーズで感情が伝わるか
- UI や背景に埋もれていないか
- 展開先が変わっても魅力が落ちないか
